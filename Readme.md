# DSA Solutions in C++

This repository contains C++ solutions for DSA problems.

## Problems Included

### 1. Two Sum

* Find indices of two numbers whose sum equals the target.
* Approach: Hash Map (`unordered_map`)
* Time Complexity: O(n)

### 2. Fibonacci Number

* Generate or find Fibonacci numbers.
* Approach: Iteration
* Time Complexity: O(n)

## Technologies Used

* C++
* STL
* Vector
* Unordered Map

## Run Program

Compile:

```bash
g++ filename.cpp -o output
```

Run:

```bash
./output
```
